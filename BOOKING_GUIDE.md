# Запись на занятия йоги

## Как это устроено

1. **Пользователь** заходит на главную страницу, в блоке «Запись на занятие» выбирает дату в календаре, видит свободные окна по времени, заполняет форму (имя, телефон, email) и нажимает «Записаться».
2. **Сайт** сохраняет запись и отправляет уведомление администратору в Telegram.
3. **Слоты** добавляются вручную в файл на компьютере (см. ниже).

## Добавление слотов вручную

Откройте файл `content/bookings/available-slots.json` в текстовом редакторе.

**Формат:**
```json
{
  "2025-02-03": ["10:00", "14:00", "18:00"],
  "2025-02-04": ["10:00", "18:00"],
  "2025-02-05": ["14:00", "18:00"]
}
```

- Ключ — дата в формате `ГГГГ-ММ-ДД`
- Значение — массив времени в формате `ЧЧ:ММ`

Добавьте нужные даты и слоты, сохраните файл. Сайт сразу подхватит изменения.

## Настройка уведомлений в Telegram

Чтобы получать уведомления о новых записях:

1. Создайте бота через [@BotFather](https://t.me/BotFather).
2. Узнайте свой chat_id через [@userinfobot](https://t.me/userinfobot).
3. Создайте файл `.env` в корне проекта:

```
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
TELEGRAM_ADMIN_CHAT_ID=ваш_chat_id
```

4. Перезапустите сайт (`npm run dev` или `npm run build && npm start`).

## Файлы данных

- `content/bookings/available-slots.json` — свободные окна (редактируете вручную)
- `content/bookings/bookings.json` — все записи пользователей (создаётся автоматически)
